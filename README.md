1. RESUMEN APLICACIÓN
Fernanpop es una plataforma de compraventa de artículos de segunda mano. Su objetivo es conectar a usuarios para facilitar transacciones seguras, rápidas y eficientes de bienes usados, promoviendo la economía circular.
