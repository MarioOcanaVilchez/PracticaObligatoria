1. RESUMEN APLICACIÓN
Fernanpop es una plataforma de compraventa de artículos de segunda mano. Su objetivo es conectar a usuarios para facilitar transacciones seguras, rápidas y eficientes de bienes usados, promoviendo la economía circular.

2. FUNCIONAMIENTO DE LA APLICACIÓN
¡¡¡IMPORTANTE!!! Para usar la app necesitas tener el Java SE Development Kit 25.0.1 instalado. Aqui esta el enlace para instalar la versión mas reciente: https://download.oracle.com/java/25/latest/jdk-25_windows-x64_bin.exe (sha256)

Al abrir la aplicacion te pregunta que si quieres usar un MOCK. Un MOCK es usar usuarios y objetos predefinidos para probar el funcionamiento de la aplicación, siempre que arranque la app se restablecen ya que las acciones que se hagan en el MOCK no tienen repercusion. El MOCK es interesante para probar la app. Los usuarios del MOCK son: Prueba1@, 1234, Sujeto de pruebas | Prueba2@, 1234, Sujeto de pruebas2   y hay creado un objeto que son gafas de sol.
